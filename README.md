## 💡 배운 것 정리

### Link (탭 메뉴)
```tsx
<Link to="/about">소개</Link>
```
→ 클릭하면 페이지 이동!

### navigate (버튼)
```tsx
<button onClick={() => navigate("/activity")}>
  활동 보기
</button>
```
→ 버튼 누르면 페이지 이동!

### 조건부 렌더링
```tsx
{isStudent ? (
  <p>학생입니다!</p>
) : (
  <p>학생이 아닙니다!</p>
)}
```
→ 조건에 따라 다른 내용 표시!
